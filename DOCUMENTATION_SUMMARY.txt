HOW TO USE THESE FILES
================================================================================

FOR GITHUB:
-----------
1. Upload KentScript_UNIFIED_DOCUMENTATION.md to your repo as README.md
2. Users can view it directly on GitHub with full formatting
3. It will appear as the project's main documentation

FOR PDF CONVERSION:
-------------------
Method 1: Using Print to PDF (Easy - Free)
  1. Open KentScript_Complete_Documentation.html in web browser
  2. Press Ctrl+P (Windows/Linux) or Cmd+P (Mac)
  3. Select "Save as PDF"
  4. Choose your filename and location
  5. Done! Professional PDF document created

Method 2: Using Pandoc (Pro - Requires Installation)
  1. Install Pandoc: https://pandoc.org/installing.html
  2. Run: pandoc KentScript_UNIFIED_DOCUMENTATION.md -o kentscript.pdf
  3. Creates beautiful PDF from markdown

Method 3: Using Online Converters (Free - Web-based)
  1. Visit: https://pandoc.org/try/
  2. Paste markdown content
  3. Select "Markdown" to "PDF"
  4. Download PDF

Method 4: Using VS Code Extension (Easy)
  1. Install "Markdown PDF" extension in VS Code
  2. Open KentScript_UNIFIED_DOCUMENTATION.md
  3. Right-click > Markdown PDF > Export PDF
  4. PDF saved automatically

FOR WEBSITE:
-----------
1. Save KentScript_Complete_Documentation.html as index.html
2. Upload to your website hosting
3. Perfect for standalone documentation site
4. Already has professional styling
